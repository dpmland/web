## Development

### Requirements

- [pnpm](https://pnpm.io/)
- [deno](https://deno.land)
- [git](https://git-scm.com)
- [node](https://nodejs.org)

### Instructions

For run this web you need clone the repo and run this:

```sh
pnpm install # Install the modules
pnpm dev # Preview the website!
```

Done!
